
# Introduction to APIs in Python

Welcome to the **Introduction to APIs** lesson! This repository contains a practical introduction to working with Application Programming Interfaces (APIs) using Python's powerful `requests` library. 

In this lesson, you will learn how to send HTTP `GET` requests, handle JSON payloads, parse nested dictionary data, and work with API query parameters.
